
# Proyecto 01 Backend


##  Iniciar Servidor

```bash
npm run dev
```

## Pasos para la configuración inicial

1. Instalar dependencias:

```bash
npm install
```

2. Crear un archivo .env en la raíz del proyecto con las variables de entorno.

3. Luego puedes iniciar el servidor con:

```bash
npm run dev
```
